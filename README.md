# logs
Função pra gravar logs em banco de dados.

# Exemplo de uso:
logs('Acessou o GitHub');

# Exemplo de retorno
João acessou o GitHub em 05/09/2017 às 10:15, no ip 0.0.0.0 e na URL http://github.com
